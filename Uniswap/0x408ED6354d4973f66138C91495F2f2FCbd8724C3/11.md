> # Should Uniswap Provide Voltz with v3 Additional Use Grant? 
> 
> ## Description
> 
> Should Uniswap provide Voltz with v3 Additional Use Grant?
> 
> This is the final on-chain vote, which is being submitted by GFX Labs on behalf of Voltz.
> 
> * The [Temperature Check](https://snapshot.org/#/uniswap/proposal/0x4f42b3215c4028edd947dc69980a9b8ba837ffbe3d0caa60d999406d634c8e79) passed with >99% in favor of the proposal.

The legal text, which would provide Voltz with a v3 Additional Use Grant, can be found [here](https://gov.uniswap.org/t/consensus-check-should-uniswap-provide-voltz-with-v3-additional-use-grant/16012?u=simonj).
> 
> ## Proposal Summary
> 
> Voltz is a new DeFi primitive, bringing interest rate swaps to DeFi in a manner that is up to 3,000 times more capital efficient than alternative models. This is achieved through a variety of mechanisms, including the use of concentrated liquidity pioneered in Uniswap v3. However, since Uniswap v3 is subject to a Business Source License, Voltz is requesting an “Additional Use Grant” that would allow Voltz to use aspects of the v3 code within its codebase.
> 
> Given the significant R&D that went into v3, Voltz is offering 1% of Voltz future tokens alongside a number of non-financial benefits.
> 
> Beyond the financial benefit for Uniswap token holders, if approved, this will also represent a first for DeFi and set precedent for future Additional Use Grant requests.
> 
> ## On-chain voting
> We are now initiating the 3rd and final governance vote. 
> 
> Since license changes must be executed via the [ENS](https://ens.domains/) domain uniswap.eth (controlled by Uniswap Governance), the governance can link arbitrary text with any subdomain of the form: X.uniswap.eth. The details of how this is done can be found in the following repo (under the Licensing section), the full simulation of the proposal is done in [here](https://github.com/Voltz-Protocol/uniswap_proposal_simulation).
> 
> If passed, Voltz will be provided with an Additional Use Grant as per the terms of the Business Source License. 1% of Voltz Tokens will be provided to Uniswap Treasury following the Voltz Token Generation Event.
>
